# Day 1 - SQL Fundamentals

## Objective
Build a strong foundation in SQL by learning how to create a database, define tables, apply constraints, and insert data into a relational database.

## Topics Covered

* CREATE DATABASE
* USE DATABASE
* CREATE TABLE
* SQL Data Types
* SQL Constraints
  * PRIMARY KEY
  * NOT NULL
  * UNIQUE
  * DEFAULT
* INSERT INTO
* SELECT Statement

## Files
* day01_sql_fundamentals.sql

## Summary

Today I learned the fundamentals of SQL, including creating a database, creating tables, working with different data types, applying constraints, inserting records, and retrieving data using the SELECT statement. These concepts form the foundation for writing SQL queries and working with relational databases.
